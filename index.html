How We Proceed

We start by analyzing the project brief, identifying what the key requirements are of the client. Then we move on to doing research about the project, understanding what the project is and how we can add value to it, we analyze the site and surroundings, the program area requirements etc. Eventually through brainstorming ideas are generated and developed with time. Then we go to production. As soon as we are done with the production we move on to implementation, testing and evaluation. Problems found are then sorted and redesigned leading to the final stage in design, production and implementation.
